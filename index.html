Welcome to my LIRI Bot.


Please open your command line and use any of the following commands:


my-tweets (this will give you the 20 most recent tweets from @HelloWorldJill)


spotify-this-song {enter a song title} (this will give you some info on the song, if no song is entered, you'll see info for Purple Rain)


movie-this {enter a movie title} (this will give you some info on the movie, if no movie is entered, you'll see info for Mr. Nobody)


do-what-it-says (this is reading a command in a different file - feel free to change that file and enter your own)
